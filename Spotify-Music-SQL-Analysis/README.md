# Spotify Music Popularity Analysis Using SQL

## Project Overview

This project uses SQL to analyse Spotify track data and identify patterns in genre distribution, artist frequency, popularity, and audio characteristics.

## Dataset

The dataset contains over 100,000 Spotify tracks with variables including artist name, track name, genre, popularity, danceability, energy, acousticness, valence, tempo, and duration.

## Tools Used

- SQL
- SQLite
- DB Browser for SQLite

## Data Preparation

- Ignored the unnecessary index column
- Checked for missing values in track name, artist, and genre
- Reviewed duplicate track and artist combinations
- Verified that numeric columns such as popularity, danceability, and energy imported correctly
- Created a cleaned SQL view for analysis

## Questions Answered

- Which genres are most common?
- Which artists appear most frequently?
- Which songs are the most popular?
- Which genres have the highest average popularity?
- Which genres are the most danceable and energetic?
