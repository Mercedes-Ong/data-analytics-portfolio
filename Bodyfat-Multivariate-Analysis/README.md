# Body Composition Multivariate Analysis

## Project Overview

This project analyses anthropometric data to understand patterns in body composition using multivariate statistical techniques.

The analysis applies robust principal component analysis (ROBPCA), clustering, and nonlinear regression to explore relationships between body measurements and body density.

---

## Dataset

Bodyfat dataset containing:

• 252 observations  
• 14 anthropometric measurements  
• body density values  

The dataset includes variables such as:

• age  
• weight  
• height  
• body circumferences  

---

## Tools Used

• R  
• Robust PCA (ROBPCA)  
• K-Means Clustering  
• Nonlinear Regression  

---

## Analysis Steps

1. Data preprocessing and standardisation  
2. Robust PCA to reduce dimensionality and detect outliers  
3. K-Means clustering to identify groups with similar body composition  
4. Nonlinear regression to model body density  

---

## Key Findings

• Several anthropometric measurements strongly influence body density  
• PCA revealed major components explaining variation in body composition  
• Clustering identified distinct body composition groups  
• Regression analysis showed nonlinear relationships between PC1 and body density
